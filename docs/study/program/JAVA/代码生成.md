# 代码生成

### Mapstruct

MapSturct 是一个生成类型安全，高性能且无依赖的 JavaBean 映射代码的注解处理器（annotation processor）。

- 注解处理器
- 可以生成 JavaBean 之间那的映射代码
- 类型安全，高性能，无依赖性
