# 概述

一种虚拟技术。

### windows/linux docker

- windows docker 宿主机 可以运行 windows 和 linux 容器;
- linux docker 宿主机只能运行 linux 容器;
- windows 宿主机可以运行 linux 容器的原因是： windows 在后台创建了一个 Linux 子系统(WSL)，因此 Linux 容器仍在 Linux 上运行;
